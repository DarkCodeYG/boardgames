## 완료된 것 (Phase 0 + 1 + 1.5)

### 엔진 (game-engine.ts)
- ✅ 게임 초기 상태 생성 (2-4인, 자원·방·가족 초기화)
- ✅ 일꾼 배치 (`placeWorker`) + 자원 획득
- ✅ 라운드 진행 (라운드 카드 공개, 자원 누적, 일꾼 회수)
- ✅ 수확 처리 (밭 수확 → 식량 공급 → 번식)
- ✅ 밭 갈기 / 씨 뿌리기 (밀=3토큰, 채소=2토큰)
- ✅ 방 건설 / 집 개조 (나무→흙→돌)
- ✅ 외양간 건설
- ✅ 가족 늘리기 (방 유무 조건, 5라운드 조건)
- ✅ 가축 시장 (양+음식1 / 멧돼지 / 소+음식1 선택)
- ✅ 목장 용량 계산 (셀×2 + 외양간 ×2)

### 농장 엔진 (farm-engine.ts)
- ✅ BFS 목장 자동 계산 (63개 테스트 통과)
- ✅ 울타리 배치 → 목장 구획 자동 인식

### 점수 엔진 (scoring-engine.ts)
- ✅ 12개 카테고리 전체 (46개 테스트 통과)

### UI 컴포넌트
- ✅ FarmBoard: 3×5 그리드 + 울타리 오버레이
- ✅ FarmCell: 방/밭/외양간/목장 시각화 + 가족 구성원 토큰
- ✅ ActionBoard: 행동 공간 + 일꾼 배치 (체스 방식: 선택→녹색●표시→클릭)
- ✅ ResourcePanel: 자원 현황 (한국어 용어 통일)
- ✅ GamePage: 다른 플레이어 탭, 배너 메시지, 라운드 진행 버튼

### 카드 데이터
- ✅ A/B덱 직업 24+23장 (이름 DB 완성, 효과 함수 미구현)
- ✅ A/B덱 소시설 24+24장 (이름+비용 DB 완성, 효과 함수 미구현)
- ✅ 주요설비 10개 (비용·효과 기본 구조 완성)
